# 老板使用手册 —— 句读授权中心

> 这份手册不需要任何技术背景。照着做就行。

## 一、这个软件是干什么的？

你是**授权管理员**。员工的电脑上都装了「句读」，但它不是装上就能用的——
必须由你发一把"钥匙"（许可证密钥）给它，并且它要定期向你电脑上的
「许可证服务器」"报到"（每 5 分钟一次，叫心跳）。

打个比方：你是门卫室，员工的软件是小区住户。住户进门要出示你发的门卡（激活），
门卡每隔一段时间要到门卫室刷一次（心跳）。你在门卫室里可以随时
**发新卡、挂失（撤销）、冻结（暂停）、延期（续期）**。

你电脑上的 `句读授权中心.exe` 就是那个"门卫室"。

## 二、安装（只做一次）

1. 在电脑上建一个**固定位置**的文件夹，比如 `D:\句读-授权\`。
   - ⚠️ **不要放在桌面**，也不要放在下载目录。因为软件第一次运行时，
     会在 exe **旁边**生成一个叫 `.data` 的文件夹，桌面会越来越乱，而且容易误删。
2. 把 `句读授权中心.exe` 复制进这个文件夹。
3. `.data` 是什么？—— **授权系统的命根子**。里面有两样东西：
   - **密钥**（相当于印章）：所有发出去的许可证都靠它盖章验证；
   - **数据库**（相当于登记簿）：你发过哪些许可证、哪些电脑在用、什么时候到期。
   - `.data` 丢了 = 登记簿和印章全丢，只能全部重来（详见"注意事项"）。

## 三、首次使用

1. 双击 `句读授权中心.exe`。
2. **第一次会弹出蓝色的"用户账户控制"窗口，问你是否允许更改——点「是」。**
   这是在给 Windows 防火墙开一扇门（放行 8768 端口），不放行员工的电脑就连不上你。
   只需要点这一次；以后每次启动它会自己检查，已经放行就不再弹了。
   （如果不小心点了"否"：员工会连不上。别慌，退出软件重新双击，再点"是"即可。）
3. 接着会出现一个**黑色窗口**（里面滚动着日志文字），并且自动打开一个网页
   —— 这个网页就是**管理页**，地址是 `http://127.0.0.1:8767/`。
   - ⚠️ **黑色窗口不能关！** 关了 = 门卫室下班，员工报到没人受理。
     用的时候让它最小化待在任务栏就行。
4. 以后想再打开管理页：直接双击 exe（如果已经在运行，就用浏览器打开上面那个网址）。

管理页顶部有一排信息，最重要的两个：

- **员工连接地址**：形如 `http://192.168.1.50:8768`，旁边有「复制」按钮——
  这就是要发给员工的地址。
- **DEV 密钥**标记：开发联调阶段会显示，正式使用请按技术人员的指示切换正式密钥。

## 四、日常三件事

### 1. 发许可证（新员工要用软件时）

1. 点顶部标签「**创建**」。
2. 填写：
   - **名称**：写员工名字或用途，比如"张三-市场部"（只给自己看）；
   - **有效期模式**：默认"有效天数"（从员工第一次激活那天算起），填 365 天即可；
     也可以选"固定截止时间"，到某个日期统一到期；
   - **最大设备数**：这张许可证允许几台电脑用。一人一机就填 1。
3. 点「**创建许可证**」。
4. 弹出窗口显示"许可证创建成功"，里面有完整的许可证密钥
   （形如 `AIREV-XXXX-XXXX-XXXX`）。
   - ⚠️ **完整密钥只显示这一次！** 请立刻点「**复制 Key**」，
     通过微信/邮件发给员工，或者先存到自己的备忘录。关掉窗口后系统里只剩
     打码的 `AIREV-XXXX-****-****`，谁也拿不回完整密钥，只能重新创建一张。

### 2. 看状态（谁在用、正不正常）

点顶部标签「**许可证**」，列表里每张许可证能看到：

- **状态**：待激活 / 已激活 / 已暂停 / 已撤销 / 已过期；
- **设备**：已绑定几台电脑（点进去能看到设备名、最近在线时间）；
- **最近心跳**：这台电脑最后一次"报到"的时间。如果很久没报到，
  说明那台电脑没开机或没连网；
- 页面顶部的「**近 10 分钟活跃客户端**」能一眼看到谁正在用。

### 3. 管理（出事时用）

在许可证列表每行右侧「操作」列：

| 按钮 | 什么时候用 | 效果 |
|---|---|---|
| **续期** | 许可证快到期了 | 加天数（有 +7 天/+30 天快捷按钮），员工端下次心跳自动拿到新期限，**员工什么都不用做** |
| **暂停** | 临时冻结（如员工休假、有争议待核实） | 员工软件锁定但记录保留，点「恢复」即可解冻 |
| **恢复** | 解除暂停 | 员工在软件锁定页点「重新验证」，或重启软件即可恢复 |
| **撤销** | 员工离职、密钥泄露 | **永久作废**，员工软件下次心跳后锁定，不可恢复（只能发新证） |
| **解绑** | 员工换电脑 | 解除旧电脑的绑定，员工在新电脑上重新激活即可，不占用新的许可证 |
| **重置全部设备** | 想清空这张证的所有设备 | 所有设备都要重新激活 |
| **编辑** | 改名称/备注 | 只改显示信息，不影响使用 |

⚠️ 撤销/暂停**不是立即生效**：员工软件每 5 分钟报到一次，**最多 5 分钟后**才会锁定。

## 五、把什么发给员工？

两样东西，缺一不可：

1. **员工连接地址**：管理页顶部，点「复制」（形如 `http://192.168.1.50:8768`）；
2. **许可证密钥**：创建时复制的那串 `AIREV-XXXX-XXXX-XXXX`。

微信/钉钉发给员工就行（密钥不算高度机密，泄露了你可以随时撤销再发新的）。

## 六、注意事项（重要！）

### ⚠️⚠️⚠️ 千万别做的事

1. **不要在有员工使用时删除 `.data` 文件夹** —— 等于烧掉登记簿+印章，
   所有已发许可证全部作废，只能全员重新激活。
2. **不要点「设置 → 危险操作 → 重新生成密钥对」** —— 等于换印章，
   所有员工端立刻验证失败，必须**全员重新激活，并且员工安装包要重新打包**。
   除非技术人员明确让你点，否则永远别点。

### ⚠️⚠️ 备份（每周花 1 分钟）

`.data` 文件夹就是整个授权系统。**每周一次**：

1. 关掉黑色窗口（停掉服务器）；
2. 把整个 `.data` 文件夹复制到 U 盘或网盘；
3. 重新双击 exe 启动。

**换电脑怎么搬**：把 exe 和 `.data` 一起拷到新电脑同一位置，双击运行即可。
员工那边**原密钥仍然有效、不用换**，但软件里记的还是旧地址，要改一下：
让员工在软件的「设置 → 许可证」里点「**退出许可证**」，回到激活页后填
**新电脑的地址和原密钥**重新激活即可（设备之前绑定过，重新激活不会多占
设备名额；操作步骤和员工手册"首次激活"一节一样）。

### ⚠️ 其它要知道的

- **你的电脑关机时**：员工已经激活的软件**照常使用**（一直能用到许可证到期日），
  但撤销/续期/暂停要等你开机、软件报到后才生效（最多延迟 5 分钟）。
- **建议给你的电脑设固定 IP**：找技术人员在路由器里把你电脑的 IP 绑定
  （叫"DHCP 静态绑定"）。否则 IP 变了，所有员工都要改服务器地址。
- **撤销有延迟**：点完撤销，员工**最多 5 分钟后**才被锁定，不是立刻。

## 七、故障自查表

| 现象 | 先检查 | 怎么办 |
|---|---|---|
| 员工连不上服务器 | ① 黑色窗口还在吗？② 你的电脑和员工电脑在同一网络吗？③ 你的 IP 变了吗？ | 窗口没了就重新双击 exe；IP 变了就把新的"员工连接地址"发给员工；还不行找技术人员查防火墙 |
| 管理页（127.0.0.1:8767）打不开 | 黑色窗口还在吗？ | 重新双击 exe；窗口里如果写着端口被占用，见下行 |
| 启动时提示端口被占用 | 是不是已经开了一个？（任务栏找黑色窗口） | 关掉重复的那个；确实被别的软件占了，找技术人员用环境变量换端口（`AI_REVIEW_LICENSE_EMPLOYEE_PORT` / `AI_REVIEW_LICENSE_ADMIN_PORT`） |
| 员工说密钥无效 | 密钥是否输错（注意没有字母 O、I、L）？ | 让员工直接复制粘贴，不要手打 |
| 员工说"已达设备数量上限" | 这张证绑了几台设备？ | 在许可证详情里把旧设备「解绑」，或把最大设备数调大 |
| 不小心点了"重新生成密钥对" | — | 全员需要重新激活；员工安装包也要重打包，立即联系技术人员 |

## 八、联系技术支持前请准备

1. 黑色窗口里的**最后几行文字**（拍照或复制）；
2. 你做了什么操作、出现了什么提示（截图）；
3. 管理页「**事件**」标签页里最近的失败记录（截图）；
4. 如果是员工端问题：员工软件里"设置 → 许可证"页面的截图。
