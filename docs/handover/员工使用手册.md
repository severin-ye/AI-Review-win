# 员工使用手册 —— 句读（许可证篇）

> 这份手册不需要任何技术背景。照着做就行。

## 一、这个软件是干什么的？

「句读」是公司内部使用的文档审校软件。它不是装上就能用的——
需要管理员（老板）发给你一把"钥匙"：**许可证密钥**。
你的软件每隔 5 分钟会自动向管理员电脑"报到"一次，确认钥匙还有效。
这些都在后台自动完成，**你正常用软件就行，基本感觉不到它的存在**。

## 二、安装（只做一次）

1. 双击安装包「**句读 Caret Setup 0.1.0.exe**」。
2. 安装过程中可以点「浏览」改安装目录（默认即可，不懂就直接下一步）。
3. 装完桌面会出现「**句读 Caret**」图标，双击打开。

## 三、首次激活（只做一次）

打开软件后，会先看到「**激活许可证**」页面。按顺序做：

1. **填"服务器地址"**：把管理员发给你的地址原样填进去，
   形如 `http://192.168.1.50:8768`（建议直接复制粘贴）。
2. **点「连接测试」**：
   - 看到**绿色提示**"连接成功"（下面还会显示服务器时间）→ 继续下一步；
   - 看到**红色提示** → 先别激活，看下面的"故障自查表"。
3. **填"许可证密钥"**：把管理员发的那串 `AIREV-XXXX-XXXX-XXXX`
   **直接复制粘贴**进去（不要手打，容易看花眼）。
4. **"设备名"可以不改**（默认用你这台电脑的名字，方便管理员识别是谁）。
5. 点「**激活**」→ 稍等一两秒，自动进入主界面，搞定。

以后每次打开软件都**不用再激活**，它会自己完成检查。

## 四、日常使用

- **什么都不用管**。激活一次管到到期日。
- 想看自己的许可证状态：左侧菜单「**设置**」→ 往下找到「**许可证**」卡片，能看到：
  状态、许可证编号、激活时间、**截止时间**、**剩余时间**、最近成功心跳、服务器状态。
- 卡片上还有两个按钮：
  - 「**立即验证**」：手动向服务器报到一次（管理员刚给你续期/恢复后点它，立刻生效）；
  - 「**退出许可证**」：清除本机的激活状态。**没事别点**，点了要重新输入密钥激活。

## 五、你会看到的提示以及怎么办

| 提示（真实文案） | 长什么样 | 含义与怎么办 |
|---|---|---|
| **暂时无法连接许可证服务器。当前许可证仍可使用至 ××××，恢复连接后将自动重新验证。** | 主界面顶部**黄色横幅**（可点 ✕ 关掉） | 管理员电脑没开机或网络不通。**不影响使用**，只要许可证没到期就照常干活，等管理员开机后自动恢复 |
| **许可证即将到期：剩余 3 天 / 1 天 / 1 小时 ……请及时联系管理员续期。** | 主界面顶部**橙色横幅** | 快到期了。截图发给管理员，提醒续期。续好后**你什么都不用做**，下次心跳自动更新 |
| **许可证已到期，请联系管理员续期。** | 软件回到激活页/提示页 | 找管理员续期。续好后**重启软件**即可（启动时会自动向服务器同步新期限），也可以在激活页重新输入原密钥点「激活」。**不用重装软件** |
| **许可证已被管理员撤销，软件无法继续使用。请联系管理员重新获取授权。** | **锁定页**（⛔ 图标，无法进入主界面） | 你的钥匙被作废了，联系管理员。页面上有三个按钮：「重新验证」（再试一次）、「返回激活页」（换新密钥）、「安全退出」（关闭软件） |
| **许可证已被管理员暂停，软件暂时无法使用。** | 锁定页 | 被临时冻结，联系管理员恢复后点「重新验证」 |
| **已达设备数量上限，请先在管理端解绑其他设备。** | 激活时红色错误 | 这张许可证绑的电脑满了。让管理员在管理端把你不用的旧电脑"解绑" |
| **检测到系统时间异常回拨……** | 锁定页 | 你的电脑系统时间被人为往回拨过。把系统时间调正确，然后重启软件 |

## 六、注意事项

1. **许可证绑定你这台电脑**。换电脑想用：让管理员在管理端"解绑"你的旧电脑，
   然后在新电脑上重新走一遍激活流程。
2. **不要把系统时间往回拨**（比如为了延长试用期改日期）——会触发安全检测，
   软件会锁定并要求你校准时间。
3. **软件卸载重装不会丢激活**：许可证凭证存在系统的用户目录里，不在安装目录，
   重装后打开还是激活状态。但**换电脑不行**（见第 1 条）。
4. 管理员电脑关机期间，你的软件**照常用**（直到许可证到期日），不用担心。
5. 许可证密钥不用背，但也别随手发给外人。万一泄露了，管理员可以撤销重发。

## 七、故障自查表

| 现象 | 最可能的原因 | 怎么办 |
|---|---|---|
| 连接测试红色失败 | ① 服务器地址填错 ② 管理员电脑没开机 ③ 你和管理员不在同一网络 | 核对地址（注意是 `http://` 开头、端口号 8768）；问管理员"门卫室"开了没有；连公司内网/同一个 Wi-Fi |
| 激活提示"许可证密钥无效" | 密钥输错（字符里没有 O、I、L，容易看花眼） | 删掉重新**复制粘贴**，别手打 |
| 激活提示"许可证已被管理员暂停/撤销" | 这张证被管理员冻结或作废了 | 联系管理员确认，换发新证 |
| 激活提示"已达设备数量上限" | 这张证绑满了电脑 | 让管理员解绑旧设备 |
| 激活提示"本机时间与服务器偏差过大" | 你的系统时间不准 | 右键任务栏时钟 → 调整日期/时间 → 打开"自动设置时间"，再重试 |
| 激活提示"无法连接许可证服务器/请求超时" | 同"连接测试失败" | 同上①②③ |
| 黄色"无法连接服务器"横幅一直在 | 管理员电脑一直关着 | 不影响使用；如果持续好几天，提醒管理员开机 |
| 突然弹出锁定页 | 被撤销/暂停，或系统时间被改过 | 按锁定页上的按钮操作：先「重新验证」，不行就联系管理员 |

## 八、还是搞不定？

截图这三样东西发给管理员或技术支持：

1. 你看到的**完整提示页面**（激活页红字/锁定页/黄色横幅）；
2. 「设置 → 许可证」卡片的内容；
3. 管理员发你的服务器地址和密钥前 8 位（形如 `AIREV-XXXX`，**不要发完整密钥**）。
